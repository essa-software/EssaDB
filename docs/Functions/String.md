# String functions

## `ASCII`

Returns first byte in the UTF-8 representation of the given string.

- Arguments: `s`: string
- Returns: `s[0]` casted to int. If `s[0] > 127`, return `s[0] - 128`.
- Throws: If `s` is empty string
